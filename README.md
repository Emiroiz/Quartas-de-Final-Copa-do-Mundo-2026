# 🏆 Quartas de Final - Copa do Mundo 2026

Projeto desenvolvido em Python para realizar sorteios e simulações relacionadas às quartas de final da Copa do Mundo de 2026.

## 📖 Sobre o Projeto

Este programa foi criado com o objetivo de praticar conceitos de programação em Python através de um tema esportivo: a Copa do Mundo.

A aplicação permite realizar sorteios de seleções classificadas para as quartas de final, tornando o processo divertido e aleatório.

## 🚀 Tecnologias Utilizadas

* Python 3

## 📂 Arquivos do Projeto

* `Quartas copa do Mundo.py` - Simulação das quartas de final.

## 🎯 Objetivos de Aprendizagem

* Trabalhar com variáveis e tipos de dados.
* Utilizar estruturas de decisão.
* Aplicar estruturas de repetição.
* Manipular listas.
* Utilizar funções da biblioteca padrão do Python.
* Desenvolver lógica de programação.

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/Emiroiz/Quartas-de-Final-Copa-do-Mundo-2026.git
```

2. Entre na pasta do projeto:

```bash
cd Quartas-de-Final-Copa-do-Mundo-2026
```

3. Execute o arquivo desejado:

```bash
python "Quartas copa do Mundo.py"
```

## 💡 Exemplo de Uso

O usuário executa o programa e recebe um sorteio aleatório das seleções classificadas para as quartas de final da Copa do Mundo 2026.
